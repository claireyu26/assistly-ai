# Public Assets Directory

This directory contains static assets that are publicly accessible.

## Required Files

### sms-preview.png
- **Location**: `/public/sms-preview.png`
- **Purpose**: SMS notification preview image for the Product Preview section on the landing page
- **Recommended Size**: Mobile phone screenshot aspect ratio (9:16)
- **Format**: PNG with transparency preferred
- **Usage**: Displayed in the Product Preview section before the footer on the main landing page

## How to Add the Image

1. Place your SMS preview image in this directory as `sms-preview.png`
2. The image should show an example of an SMS notification/reminder from Assistly AI
3. The image will automatically be optimized by Next.js Image component
